新疆游记——北疆大环线

发表于2024-07-13|更新于2024-07-14|旅行

Abstract

我们的行程安排参考了xhs上某一篇北疆大环线的地图：

针对该行程大纲，我们进行灵活的调整，最后计划表如下：

天数 安排
-2,-1 Z40次动车，39小时
0 乌鲁木齐休整，闲逛
1 乌鲁木齐 => 天山天池 => 阿勒泰
2 阿勒泰 => 禾木
3 禾木 => 喀纳斯
4 喀纳斯 => 乌尔禾
5 乌尔禾 => 魔鬼城 => 赛里木湖
6 赛里木湖 => 特克斯
7 特克斯 => 琼库什台
8 琼库什台 => 蜜蜂小镇
9 蜜蜂小镇 => 百里画廊 => 独库公路 => 乌鲁木齐
10 各回各家

Before Start

  • 硬卧

    第一次坐硬卧，Z40的硬卧还挺难抢；我睡中铺，底下有小孩买了硬座来睡大人的卧铺非常吵，我拿AirPods当耳塞的效果还不错。

  • 租车

    租了辆7座商务车（别克GL8），差点没放进6个人的行李，主要是不能带的非常大，后备箱的厚度有限；在新疆玩了这么多天，看到的坦克300着实是多，下次5人以下的话我就租坦克300玩玩了。

  • 租相机

    在内啥租了个SONY a6700配18-135 3.5-3.6的镜头，租13天花了600多，性价比其实还行；问邱老师借了个三脚架，主要是想用来拍星轨和银河。

    在第三天发现拍出的照片有黑点，排查了一下发现是CMOS上的脏东西，因为没什么工具擦，影响也不是太大，就没管了，所以租来的时候要多多检查问题。

  • 行李

    我感觉这次旅行带非常有用的东西：防晒霜、防晒衣、一次性的各类用品、折叠衣架、斜挎小包、墨镜、帽子、防晒口罩（带着闷气，不是很喜欢）、冲锋衣（带内胆）

行云流水

Day0

早上8点多到达乌鲁木齐，取完车9点多，逛了下国际大巴扎和领馆巷，感觉一般，可以买点酸奶和水果吃；午饭吃了大盘鸡；下午去新疆博物馆学习了一下，里面有些展馆和新疆的关系真心不大，有关于铜镜、服饰妆容、干尸墓地等等内容，我了解颇少

乌鲁木齐的路还挺难开的，车道有些真的窄，路上的车开的也很野；新疆的日落要10点以后，某种程度上迎合了我本来的作息hhh

Day1

开车60km左右去了天山天池，栈道很像刚去过的云杉坪，湖水也是蓝绿色的和蓝月谷很像。

下午开了500km左右来到阿勒泰，晚上吃了一个韩式烤肉，但肉质确实非常烂，怎么评上阿勒泰top4烧烤的？

Day2

吃了阿勒泰特色的一个早餐，叫努尔古丽奶茶风味餐厅，服务员会拿菜来让你选要不要，这里面马肠非常贵，200g70块钱，一共花了168块，血亏。

接着是一路开车走国道和非常曲折的山路来到禾木，本来打算住布尔津，但是车程太长，临时定了禾木的便宜酒店，本来以为会很差但还行；禾木实行三级管理，门口停车场没满的话就停门口坐大巴进去；满了就自己开车去滑雪场的停车区；晚上超过9点之后就直接去禾木；禾木有个日落观景台，虽然10:30才日落，但是太阳会提前被山挡住所以没看到日落。当天晚上也没看到星星，什么时候能拍到银河啊。

Day3

从禾木绕曲折的山路去喀纳斯，喀纳斯进景区要做好久的大巴，我们今天安排要去乌尔禾，时间很紧张，所以只随便看了下，午饭匆匆解决；其实喀纳斯的景色比禾木好不少，留宿喀纳斯是个不错的选择。

还有神仙湾、月亮湾等等地方，景色也都非常不错。逛完喀纳斯就去乌尔禾了，车程太远，不可避免地走了180km左右的夜路，将近1点才到民宿。途中有个非常逆天的国道or省道or县道，正在建设中，全是泥路，对我们轮胎是个大考验。

这一天路中，轮胎胎压过低，去修理厂一问，师傅说是轮胎磨损很严重，而且有一部分钢筋都断了，然后就换了个胎。

Day4

在乌尔禾本来想去魔鬼城，然而最后选择去艾里克湖附近看戈壁和捡石头，大太阳下几个人低头全在捡石头真的挺好笑的，虽然不值几个钱，反正就是图个开心。路上还遇到一个老大爷和我们炫耀他捡的一块好石头。

捡完石头去去赛里木湖，也开了很久，路上不留神把油箱开报警了，到了加油站还下大雨停电了，不过等待过程中看到了双彩虹，非常值得！当天晚上看到很多很多星星，后面再放图。

Day5

这天对赛里木湖进行了一个环湖自驾，湖边的景色真的很美，但是湖周边的景点都太多了没时间一个个逛，环湖本身也要100km左右耗时很久，我们还得赶去其他地方所以玩的不是很细致。我非常想去湖里坐船玩，可惜那天风太大了，估计身上要湿，所以没去。

开车过程中，途径多次赛里木湖的果子沟大桥，景色也非常美，好像连霍高速也是经过这个大桥的。

由于我昨天拍星星太晚了这一天开车不多，然而在我睡觉时有人撞到了路边石墩，最后为这一撞赔了4k元…

这天晚上住在了特克斯，特克斯是一个红绿灯很少的地方，一开进去给我一种明显的外地人车感。

晚上被zzh拉去拍星星，开到了一个非常偏僻的地方，又黑又颠簸，真给我整怕了，停下来拍了星空，这一次连拍了120张，并且学会了堆栈，成功出现了星轨，但是没对准北极星，没形成圆环，而且杂星太多，应该得先减少量度不高的星星。

Day6

这天我们从特克斯开车去琼库什台住“蒙古包”（其实应该叫哈萨克族毡房），山路极其之绕，路又窄连黄虚线都没有，很多时候会车只能一半轮胎进两侧的石子里去，还有好多好多泥路，真的难开。

中途我们从侧面进入，看到了喀拉峻景点的鳄鱼湾，其实这里就是前天晚上拍星星的地方，但是晚上太黑了啥也看不到，完全是两种景色。

哈萨克族毡房是一整个大床铺睡6个人，但每个人都各自的被子和床铺，还算可以。我晚上进行最后一次观星，终于是拍到北极星了，可惜只拍了60张堆栈，还不够多，因此星轨拉得不够长。

Day7

正式第七天，早上睡了个小懒觉，早饭点了个一斤羊肉火锅，结果一堆骨头，都没多少肉，很坑。12点左右从琼库什台下山，又是非常曲折的山路，基本是原路返回特克斯，然后再开到到蜜蜂小镇，买了瓶蜂蜜带回去养生。蜜蜂小镇的宾馆半夜停电非常弱智。

Day8

正式第八天，从蜜蜂小镇出发，途径百里画廊、独库公路，终于到达乌鲁木齐！

百里画廊指的其实是100km左右的国道，他周边的景色着实非常美，这是进入独库公路回乌鲁木齐的必经之路。

独库公路好像成了一个证明车技实力的地方，为什么说独库公路车祸多，我一进入就有车硬借道超车别我，我反手也把它超了。刹车磨的声音是挺响的让我有点不敢刹，一路上看到了3起车祸。独库公路急弯很多，我其实过弯速度不慢，没有慢悠悠开，虽然不是非常稳，但是挺安全挺迅速的，也算是开过独库公路了。

After

相机拍的照片即时性挺差的，后续我懒得弄了把照片全传到网盘里让大家自取，然后还把镜头盖给丢了，立马买了一个原装的补上。租相机的主要目的是感受一下是否需要自己购买个相机，索尼的使用体验还可以，就是屏幕太差了，得看取景框。在拍摄过程中学到了不少摄影知识，希望以后能慢慢提高审美，拍出更好的照片。

我们租车买了全险，但是出事情的时候一定要及时联系租车公司！赛里木湖环湖的时候我在车上睡觉，车撞到了路边凸起的一个石墩子，把本来就有旧伤的保险杠撞得有点掉了。我看了下貌似还好，就没太当回事儿，没和租车公司说。结果还车的时候就说非常严重，需要整个换掉。因为事故发生时没有找交警备案，没法出险，这换保险杠的钱我们承担。租车公司说修理厂开价6k元（我不懂为什么换个保险杠要6k，他们给的理由是新疆修车贵），最后租车公司帮我们承担了2k，我们出4k。所以其实全险等于没买，还倒贴过去4k，这本质原因就是没及时报备，我有很大责任。

去的时候坐的硬卧，回来本来想硬座的，但是最后几天出现了超级便宜的机票，我不买实在是对不起航空公司，最后还是飞回来的。有个朋友硬座火车临时取消，还一大早白跑一趟，太幽默了hhh

新疆行作为最后的毕业旅行，着实期待了很久，也是最后一次和这些好兄弟们出去旅游了，祝我们都一路顺风。
