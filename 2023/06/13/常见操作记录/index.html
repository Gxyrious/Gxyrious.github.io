git

参考：https://blog.csdn.net/qtiao/article/details/97783243

  1. 初始化：git init
  2. 查看历史：git log
  3. 查看状态：git status
  4. 查看区别：git diff
  5. 暂存文件：git add *.java
  6. 取消暂存：git rm xxx
  7. 提交：git commit -m xxx
  8. 查看分支：git branch
  9. 创建分支：git branch xxx
  10. 切换分支：git checkout
  11. 创建分支并切换：git checkout -b xxx
  12. 合并分支：git merge
  13. 拉取/推送仓库：git pull/push
